# 🚀 Enhanced Todo App with Docker

A feature-rich, containerized Todo application built with Flask and SQLite, fully dockerized with Docker Compose.

## ✨ Features

### Core Features
- ✅ **Create Tasks** with titles, descriptions, priorities, categories, and due dates
- 🎯 **Priority Levels**: High 🔴, Medium 🟡, Low 🟢 with color-coded cards
- 🏷️ **Categories/Tags**: Organize tasks by custom categories
- 📅 **Due Dates**: Set deadlines and see upcoming tasks
- 🔍 **Search**: Find tasks quickly with keyword search
- 🎨 **Filters**: Filter by priority, category, and completion status
- ✏️ **Edit Tasks**: Modify existing tasks anytime
- 📊 **Statistics Dashboard**: Real-time task statistics
- 🔌 **RESTful API**: Programmatic access to all todo operations
- 🐳 **Dockerized**: Easy deployment with Docker Compose
- 💾 **Persistent Storage**: Data persists across container restarts
